Savoreat

place to store restaurant order history + see what was good last time


BASE FEATURES:
- account system
- restaurants in proximity
- for each restaurant
	- pull menu from restaurant
	- contains dish tried + rating + historical orders

ADVANCED FEATURES:
- social elements
	- what most people have tried/rated well
	- how many times people have ordered certain items
- order via Grubhub/DoorDash/etc.
- recommendation engine
- receipt scanning + letter recognition to automate historical orders
