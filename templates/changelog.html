Version 2.0

updated March 20th, 2021
  • Added the "country" field for querying.
  • Fixed an issue in which ChartsJS included players from countries that were not part of the original query.
  • Added overall head-to-head for country queries only.

Version 1.0

updated March 19th, 2021
  • Placed an upper bound on number of search results to increase loading speed. Any queries with over 2,500 rows will now be capped to 2,500.
  • Added first names to player profiles.
  • Fixed an issue in which empty player rankings throw errors. They now show "(NA)"s instead.
  • Fixed an issue in which secondary links did not load properly.
  • Added some design elements (horizontal line breaks, etc.).
