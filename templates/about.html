RallyRecords is a custom query builder for tennis match data.

You can use it to search for:
  • all matches on a particular surface.
  • all matches where your favorite player has played.
  • a head-to-head comparison between two players.
  • a particular location.
  • any combination of the above variables!
The available data is updated periodically (not continuous, unfortunately!), so the most recent matches may not be immediately available. We have data going back to 2005 (ATP) and 2007 (WTA).
RallyRecords is built on a Python backend (Flask) and a MongoDB cloud cluster. It uses Jinja2 as the templating engine and ChartsJS for the beautiful, dynamic graphics. It exists on the web thanks to Heroku.
